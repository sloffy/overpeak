# Инструкция по тестированию Frontend

## Шаг 1: Запуск базы данных PostgreSQL

Откройте терминал в корне проекта и выполните:

```bash
docker-compose up postgres -d
```

Проверьте, что контейнер запущен:
```bash
docker ps
```

Вы должны увидеть контейнер `postgres-overpeak` в списке.

## Шаг 2: Запуск приложения

В том же терминале (или в новом) перейдите в папку `overpeak` и запустите приложение:

```bash
cd overpeak
mvn spring-boot:run
```


Дождитесь сообщения в консоли, что приложение запущено (обычно это `Started OverpeakApplication`).

## Шаг 3: Открытие Frontend в браузере

После успешного запуска приложения откройте браузер и перейдите по следующим адресам:

### Главная страница (статистика)
```
http://localhost:8087/
```

## Остановка приложения

Для остановки приложения нажмите `Ctrl+C` в терминале, где оно запущено.

Для остановки PostgreSQL:
```bash
docker-compose down
```
